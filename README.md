## ❤️ Heart Disease Analysis Dashboard (Power BI)

This project presents an interactive data analysis dashboard built using Power BI to explore key factors influencing heart disease survival rates.

The objective of this project is to transform raw clinical data into meaningful insights through data cleaning, modeling, and visualization techniques.

---

## 🔍 Project Overview

The dashboard provides a comprehensive analysis of patient survival trends based on age groups, medical conditions, and other key health indicators.

It enables users to easily identify patterns and relationships within the data using dynamic and interactive visuals.

---

## 📈 Key Insights

- Survival rate analysis across different age groups  
- Impact of factors such as smoking, high blood pressure, anaemia, and diabetes  
- Average age of survival and mortality distribution  
- Identification of high-risk age segments  

---

## 🛠 Tools & Technologies Used

- Power BI Desktop  
- Power Query (Data Cleaning & Transformation)  
- DAX (Data Analysis Expressions)  

---

## 📊 Dashboard Features

- Interactive visuals with slicers and filters  
- KPI cards for quick insights  
- Trend analysis using line and bar charts  
- Clean and user-friendly layout design  

---

## 🚀 Outcome

This project demonstrates how data visualization and analytics can be used to uncover critical insights in healthcare data, supporting better decision-making.
